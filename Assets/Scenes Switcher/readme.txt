Scenes Switcher v1.0

Importing new version of an asset:
1) Delete an old version of the product - Assets/Scene Switcher folder
2) Import from Asset Store to Assets folder

How to launch Scene Switcher
1) Launch your project on the device
2) The floating window of Scene Switcher will show automatically in the top-left corner
3) Drag the window as you need
4) Switch scenes with the arrow buttons on the screen
